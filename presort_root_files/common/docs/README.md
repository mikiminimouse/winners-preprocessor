# Архив устаревших компонентов

Эта директория содержит устаревшие компоненты, которые были заменены новыми решениями в проекте.

## Структура архива

### preprocessing_router/
Содержит полную копию устаревшего компонента `preprocessing/router/`, который был заменен 
эталонным решением в `final_preprocessing/docprep/`.

**Дата архивации**: December 20, 2025
**Причина**: Полное дублирование функциональности в новом компоненте

### preprocessing_cli/
Содержит устаревшие CLI handlers, которые больше не используются после миграции на 
`final_preprocessing/docprep/cli/`.

**Дата архивации**: December 20, 2025
**Причина**: Замена на новые CLI компоненты

## Активные компоненты

Следующие компоненты остаются активными и необходимы для работы системы:

1. **preprocessing/sync_db/** - Синхронизация протоколов из удаленной MongoDB
2. **preprocessing/downloader/** - Загрузка документов с zakupki.gov.ru
3. **preprocessing/scheduler/** - Планирование задач
4. **preprocessing/cli/** - Интерфейс командной строки для синхронизации и загрузки

## Эталонное решение

Все функции обработки документов теперь реализованы в:
**final_preprocessing/docprep/** - Эталонный компонент обработки документов

## Восстановление

В случае необходимости компоненты могут быть восстановлены из архива:

```bash
# Восстановление router компонента
cp -r archive/preprocessing_router/* preprocessing/router/

# Восстановление CLI handlers
cp archive/preprocessing_cli/* preprocessing/cli/handlers/
```

## Лицензия

Содержимое архива распространяется под той же лицензией, что и основной проект.
